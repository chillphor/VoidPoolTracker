# VoidPoolTracker

A lightweight World of Warcraft addon designed to track your progress for the **Void-Touched Chest** and provide quick access to **Mythic+ Teleports**.

## ✨ Features

### 📦 Loot Tracking (VoidPoolTracker)
Never lose track of your loot lockout again. This addon provides a simple interface to help you visualize:
* **Remaining Items**: Clearly display which items are still available from the Void-Touched Chest.
* **Real-time Updates**: Automatically update data based on your character's current loot eligibility.

### 🌀 Mythic+ Portal Menu
Stop digging through your spellbook. This addon includes built-in utilities for Mythic+ players:
* **Quick Teleport**: A dedicated menu or set of shortcuts to instantly cast any Dungeon Teleport you have earned.

## 🛠 Installation

1. Download the latest version of the addon.
2. Extract the folder into your World of Warcraft directory: `\_retail_\Interface\AddOns`.
3. Restart the game or type `/reload` in-game.

## ⌨️ How to Use

### Commands
* `/vp` — Open the main tracking window.

### Integration
* **Minimap Icon**: Click the icon to toggle the tracking window.

## ⚙️ Technical Details
* **Memory Footprint**: Extremely low (~20KB), optimized for performance during combat.
* **Localization**: Currently supports English and **Simplified Chinese**.
* **Dependencies**: None. This is a standalone addon and does not require any additional libraries.

## 📧 Support & Feedback
If you encounter any bugs or have suggestions for new features, please submit an **Issue** on the GitHub repository to contact the author.

> **Note**: This addon is currently in active development. Please ensure you are using the latest version before reporting issues.

# VoidPoolTracker (晦暗虚空宝箱追踪器)

一款专为《魔兽世界》设计的轻量化插件，旨在帮助玩家追踪 ** 晦暗虚空宝箱** 的掉落进度，并集成了便捷的 **大秘境传送门** 管理功能。

## ✨ 核心功能

### 📦 宝箱掉落追踪 (VoidPoolTracker)
再也不用担心忘记拾取进度。本插件提供简洁的界面，让您直观查看：
* **剩余掉落物品**：清晰显示宝箱中还有哪些物品尚未获取。
* **实时状态更新**：根据角色当前的拾取资格自动更新数据。

### 🌀 大秘境传送门菜单
告别翻阅法术书的烦恼。插件为大秘境玩家提供了内置实用工具：
* **快捷传送**：专用的菜单或快捷键，一键开启您已获得的所有副本传送门。


## 🛠 安装方法

1. 下载最新版本的插件。
2. 将文件夹解压至您的魔兽世界安装目录：`\_retail_\Interface\AddOns`。
3. 重启游戏或在游戏内输入 `/reload`。

## ⌨️ 使用说明

### 常用指令
* `/vp` — 打开掉落追踪主窗口。

### 功能集成
* **小地图图标**：点击图标切换追踪窗口


## ⚙️ 技术细节
* **内存占用**：极低（约 20KB），已针对战斗表现进行深度优化。
* **多语言支持**：目前支持英文与 **简体中文**。
* **依赖项**：无。这是一款独立的插件，无需安装其他前置库。
